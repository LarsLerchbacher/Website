3D Renderer in Scratch ====================== A simple 3d wireframe renderer in scratch using perspetive projection. I made it out of boredom when I was ill and couldn't do anything but ly in bed for a week. I know, I should have rested instead... The project uses the draw pen extension of scratch to draw shapes consisting of vertices defined in the 3 coordinate lists. They are connected in the order defined in the 4th list. Adding a goto instruction in the 4th list will cause the program to go to the new vertice without drawing. You can find the project here. This project was inspired by the youtuber Garbaj and his video I tricked a 2d game engine into doing 3d (without raycasting)

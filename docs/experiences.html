Operating Systems ================= - Windows 2000 Professional it runs on the first laptop I ever used On this machine I initally learned to use MS office - Windows 10 Used it for quite a while, but support for it has now ended - Windows 11 I use it for gaming, school and game development, because the engines I use(d) are installed here - Linux - Debian runs most of our servers - I tried out Ubuntu, but I don't actively use it - Linux Mint, my first contact with linux, only runs on my sisters laptop as Windows 10 support ended. Works as well as windows for her. I occasionally have to do updates for her though. - Gentoo Linux It's really the best linux for me personally. It install packages by compiling them from source (my school laptop uses a binary package host in addition to that, to speed things up) and it does not have any installer except yourself. You even have to configure and compile your own kernel. For that, I mostly use defaults, because I am not that experienced witht the linux kernel's options. It runs on my main PC and my school laptop, dual booting with windows 11 on both
